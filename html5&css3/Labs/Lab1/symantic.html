HTML5 Semantic Elements Demo

Exploring new semantic elements: <time>, <meter>, <mark>, and <progress>

1. The <time> Element

The <time> element represents a specific date, time, or duration. It helps search engines and accessibility tools understand temporal information.

Examples:

The event starts at 14:30 (2:30 PM)

The meeting is scheduled for February 15, 2024

Course duration: 2 hours and 30 minutes

Publication date: January 27, 2024

Last updated: last week

2. The <meter> Element

The <meter> element represents a measurement (gauge) within a known range, such as disk usage, temperature, or ratings.

Examples:

Disk Space Usage:

Used: (60%)

Temperature:

CPU Temp: 65°C

Student Grade:

Score: (8.5/10)

Battery Level:

Battery: 25% (Low)

Internet Signal Strength:

Signal: (4/5 bars)

3. The <mark> Element

The <mark> element highlights or marks text that is relevant in a particular context (typically highlighted in yellow).

Examples:

Search Results: The HTML5 specification introduces many new elements including semantic tags for better document structure.

Important Information: Note: All assignments must be submitted by Friday

Error Highlighting: The correct syntax is: <article> not <div>

Learning Material: The DOCTYPE declaration is required at the beginning of every HTML5 document.

4. The <progress> Element

The <progress> element represents the completion progress of a task (such as downloading, uploading, or form submission).

Examples:

File Download (50% complete):

50%

Video Upload (75% complete):

75%

Course Completion (3 out of 8 modules):

Module 3/8

Project Tasks (9 of 12 completed):

75% Done

Installation (0% - Not started):

0%

Processing (100% - Complete):

Complete ✓
